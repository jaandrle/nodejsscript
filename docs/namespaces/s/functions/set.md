# Function: set()

> **set**(`options`): `void`

Sets global configuration variables

## Parameters

• **options**: `string`

Available options:
       - `+/-e`: exit upon error (`config.fatal`),
       - `+/-v`: verbose: show all commands (`config.verbose`),
       - `+/-f`: disable filename expansion (globbing)

## Returns

`void`
