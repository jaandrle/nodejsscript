# Variable: mv

> `const` **mv**: [`MoveFunction`](../interfaces/MoveFunction.md)

Moves files. The wildcard `*` is accepted.

## Param

Available options:
       - `-f`: force (default behavior)
       - `-n`: no-clobber

## Param

The source.

## Param

The destination.

## Returns

Object with shell exit code, stderr and stdout.
