# Variable: rm

> `const` **rm**: [`RemoveFunction`](../interfaces/RemoveFunction.md)

Removes files. The wildcard `*` is accepted.

## Param

Available options:
       - `-f` (force),
       - `-r`, `-R` (recursive)

## Param

Files to remove.

## Returns

Object with shell exit code, stderr and stdout.
